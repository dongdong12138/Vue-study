v-model用法：
input 和 textarea 的用法：
{{ value }}

{{ msg }}
单选框用法：
单个单选框：用 v-bind ------
单选框用 v-model 不生效：
单个单选框：用 v-model------
多个单选框：v-model 绑定的是选中的那个单选框的value值
苹果：
桃子：
西瓜：
------ {{ checkFruit }}
复选框用法：
单个复选框：用 v-bind ------ 单个复选框：用 v-model -----
多个复选框：
如果绑定的是数组，v-model就与选中的复选框的 value 相关
如果绑定的是字符串，v-model就与选中的复选框的 checked 的布尔值相关
苹果： 桃子： 西瓜：
------选中了：{{ checkFruits }}
下拉框用法：
单选的下拉框：
------ {{ checkFruit }}

多选的下拉框：
------ {{ checkFruits }}
